# AI-Retail-Analytics-System
使用Python、YOLOv8 與 Gemini API等開發識別多人身材,給出當前多眾適合的服飾與配色,解決引入不適合自己客群的服飾,減少成本開銷。
1.業務痛點解決與數據洞察 ：
運用 YOLOv8 進行多目標客流偵測,結合 MediaPipe 實作高精度特徵萃取（臉部膚色 KMeans 聚類、動態補償之肩腰臀比例量測）,解決傳統服飾業無法精準掌握門店核心客群（體型/膚色）的痛點,將客觀影像轉化為量化特徵數據。

2.導入大語言模型自動化決策 ：
不只停留在影像辨識,串接 Google Gemini API（LLM）作為決策大腦。系統能根據偵測到的客群眾數分佈,自動生成「人類可讀、具備商業指導性」的櫥窗剪裁與季度色彩採購建議,大幅縮短行銷人員分析數據的時間。

3.數據可視化：
使用 Streamlit 獨立完成前端互動式儀表板（Dashboard）的開發與測試。運用 Pandas 架構資料流 Pipeline，將即時推論數據動態轉化為結構化 DataFrame，並實作「一鍵匯出 Excel 營運報表」功能，達成技術落地與無痛對接企業內部作業流程。
